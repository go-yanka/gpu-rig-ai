Third iteration of full corpus re-ingest. Applies ALL lessons from v2 (full chronology in
`LESSONS_2026-05-08.md` L1-L13 + `EVOLUTION_2026-05-07.md`).

## Why v3

V2 ingest reached 0.8447 G1 recall @0.95 threshold — FAIL. Diagnosis traced to:
1. **Mega-chunks** — 25 chunks > 8000 chars (max 438,533) embedded as single vectors with no semantic precision. Real fix needed in chunker.
2. **wait=False upsert ghost loss** — 2,393 doc_ids whose manifest said `upserted=1` but Qdrant didn't have. wait=True fix in place but state was already corrupt.
3. **Manifest/Qdrant divergence accumulated** across patches.

V3 applies all fixes from a clean slate so we can honestly measure the gates.
